Scenario - main menu

GIVEN I’m an app user

WHEN I run the program

THEN I see main menu

Tech notes:
Main menu consists of the next items:

(1) Sign Up

(2) Sign In / Sign Out

(3) Product Catalog

(4) My Orders

(5) Settings

(6) Customer List

When User is logged in, then paragraph 2 label is changed to “Sign Out” instead of Sign In.
